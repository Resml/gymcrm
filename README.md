# GymCRM

A modern Multi-Tenant Gym Management WhatsApp CRM. 
Built using React, Vite, Framer Motion, NestJS, and Prisma.

Features:
- **Tenant Isolation**: Independent gym spaces where a single master platform orchestrates billing and limits.
- **WhatsApp Official API Integration**: Broadcast membership reminders directly to mobile natively.
- **Premium UI**: Crafted using "Pro Max" aesthetics, Framer-motion scroll mapping, and OLED layout concepts.
